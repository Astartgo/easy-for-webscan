# easy-for-webscan
根据WebBatchRequest图形化二次开发  
1增加了指纹识别模块，可识别大概上千条指纹  
2增加了IP段处理,支持C段和B段处理  
3增加了301处理  
4增加了去重空节点的功能，勾选后扫描为空的资产可以不进行显示，可节省内存消耗,推荐勾选此选项  
5修改了在Jtable中排序后浏览器打开索引错误
